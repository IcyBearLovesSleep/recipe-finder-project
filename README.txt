recipe finder 

finds recipes based on ingredients inputted, takes in ingredients parameter as string, stored into 'foodlist', only valid ingredients tested by the 'ingredientsVal.txt' are allow to be inputted

scans for ingredients of recipes from 'recipelinks.txt', includes partial recipe web links from
allrecipes.com,
food.com,
ambitouskitchen.com,
averiecooks.com,
bbcgoodfood.com

return the number of matched recipe results with recipe title, recipe make-time, ingredients used, recipe link
